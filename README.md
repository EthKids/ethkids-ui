# ethkids-ui

## Project setup
```
npm install
```
In order to find the EthKids Solidity contracts you need to point them in 
`src/store.js`

and provide

```
registryAddress: '0x95e8e2D4D5E427f1620aF7264B6B79e2fAeFb8A0',
communityCreationBlock: 4455521,
requiredNetwork: 4,
```
