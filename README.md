# ethkids-ui

## Project setup
```
npm install
```
In order to find the **EthKids Solidity contracts** you need to point them in 
`src/store.js`

and choose/uncomment one of sections for
 
`Mainnet` 

OR

`Rinkeby`

 OR
 
 `Ropsten`
